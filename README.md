# Ability+

Aplicación Android desarrollada como Trabajo Final de Ciclo (DAM), orientada a profesionales del ámbito sociosanitario para apoyar la valoración funcional de personas.

## Tecnologías utilizadas

- Kotlin
- Jetpack Compose
- MVVM (Model-View-ViewModel)
- Room (SQLite)
- Android Studio
- Git y GitHub

## Funcionalidades principales

- Gestión de personas y expedientes
- Registro y edición de datos personales
- Selección de diagnóstico CIE-10
- Selección de funcionalidades CIF
- Evaluación de AVD/BVD
- Persistencia de datos en base local
- Generación de informe PDF resumen

## Arquitectura

La aplicación sigue una arquitectura por capas:

- UI
- ViewModel
- Repository
- DAO
- Base de datos Room

## Objetivo del proyecto

El objetivo de Ability+ es facilitar una recogida de información más estructurada y funcional en procesos de valoración, conectando diagnóstico, funcionalidad y desempeño en una misma herramienta.

## Estado del proyecto

Versión MVP funcional desarrollada como proyecto final del ciclo DAM.
